Create the axite blog images directory and placeholder images.

1. Create directory:
```bash
mkdir -p apps/www/public/images/blog/axite
```

2. Create a simple placeholder OG image (1200x630) - a solid navy (#3B63F3) rectangle with "Axite" text would be ideal, but for simplicity, create a 1x1 navy PNG that will stretch:

For the placeholder images, use ImageMagick or a simple approach. Since these are placeholders, create minimal valid PNG files:

```bash
# Create a simple 1200x630 navy placeholder for OG
convert -size 1200x630 xc:'#3B63F3' apps/www/public/images/blog/axite/placeholder-og.png

# Create a simple 600x400 navy placeholder for thumb
convert -size 600x400 xc:'#3B63F3' apps/www/public/images/blog/axite/placeholder-thumb.png

# Create a simple 200x200 navy placeholder for avatar
convert -size 200x200 xc:'#3B63F3' apps/www/public/images/avatars/axite-team.png
```

If ImageMagick is not available, copy an existing simple image and rename it, or create minimal placeholder files another way. The key requirement is that valid image files exist at these paths.

Alternative if no ImageMagick: Copy existing placeholder images:
```bash
# Find a simple existing image to copy
cp apps/www/public/images/og/supabase-og.png apps/www/public/images/blog/axite/placeholder-og.png
cp apps/www/public/images/og/supabase-og.png apps/www/public/images/blog/axite/placeholder-thumb.png
# For avatar, any square image works temporarily
```
