**Phase 9 Success Criteria (from ROADMAP.md):**
1. Header shows Axite navigation (Product, Pricing, Enterprise, Blog, Trust)

4. Navigating to old Supabase product URLs returns 404 or redirects
5. No "Supabase" text visible in navigation or UI chrome (blog content excluded)

**Verification Steps:**

1. Check header nav items:
```bash
grep -E "Product|Pricing|Enterprise|Blog|Trust" apps/www/data/nav.tsx
```

2. Check footer copyright:
```bash
grep "Axite" apps/www/components/Footer/index.tsx
```

3. Check mobile menu exists and has Axite content:
```bash
grep -E "Get started|Contact sales" apps/www/components/Nav/MobileMenu.tsx
```

4. Check old product URLs are not linked:
```bash
grep -c "/database" apps/www/data/nav.tsx apps/www/data/Footer.ts 2>/dev/null || echo "0"
grep -c "/auth" apps/www/data/nav.tsx apps/www/data/Footer.ts 2>/dev/null || echo "0"
```

5. Check for Supabase text in nav components (should only be in imports for logo):
```bash
grep -r "Supabase" apps/www/components/Nav/*.tsx apps/www/components/Footer/*.tsx | grep -v "import\|Logo\|logo" | wc -l
```
