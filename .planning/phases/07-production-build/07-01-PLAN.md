<task type="auto">
  <name>Task 2: Fix invalid Tailwind gradient class in career.module.css</name>
  <files>apps/www/styles/career.module.css</files>
  <action>
Fix the invalid Tailwind class `from-background-alternative` in career.module.css line 2.

The Tailwind config defines `background-alternative-default` but not a simple `background-alternative` variant for gradients.

Replace:
```css
@apply bg-gradient-to-r from-background-alternative via-border-strong to-background-alternative rounded-full p-[1px] aspect-square absolute;
```

With:
```css
@apply bg-gradient-to-r from-background-alternative-default via-border-strong to-background-alternative-default rounded-full p-[1px] aspect-square absolute;
```

This uses the correct `background-alternative-default` color token that exists in the Tailwind config.
  </action>
  <verify>
Run: `grep "from-background-alternative-default" apps/www/styles/career.module.css`
Verify: The corrected class name appears in the file
  </verify>
  <done>career.module.css uses valid Tailwind class `from-background-alternative-default`</done>
</task>
